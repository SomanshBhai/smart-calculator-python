# 🧮 Smart Calculator

A simple desktop calculator built using **Python** and **Tkinter**.

This project performs basic arithmetic operations through a clean graphical user interface. It was created as part of my Python learning journey to improve my programming and GUI development skills.

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🖥️ Easy-to-use graphical interface
- ⚠️ Basic input validation
- 🚀 Beginner-friendly project

---

## 🛠️ Built With

- Python 3
- Tkinter

## ▶️ How to Run

1. Install Python 3.
2. Download or clone this repository.
3. Open the project folder.
4. Run the following command:

```bash
python main.py
```
